Config

This section contains all information about the Glasscat server configure options used to compile the current executable.

Release properties

 

Release Properties are extracted from the Node.js process.release property. For more details please read the Node.js v7.0.0 Documentation.
 
Config properties

 

Environment Properties are extracted from the Node.js process.config property. For more details please read the Node.js v7.0.0 Documentation.
